# Marketing Growth Automation System

## Project Overview

Marketing Growth Automation System is a modern full-stack marketing platform designed to help businesses generate, collect, and manage leads efficiently. The project combines a responsive marketing website, Python backend, AI-powered chatbot, Google Forms lead collection, Zapier automation, and Mango CRM integration into a single automated workflow.

The platform enables businesses to capture potential customer information through an interactive website and Google Forms, automatically process the data through Zapier workflows, and store qualified leads directly in Mango CRM for follow-up and customer management.

## System Workflow

1. User visits the marketing website.
2. User interacts with the AI chatbot for assistance.
3. User submits details through Google Form.
4. Google Form triggers a Zapier automation.
5. Zapier processes the submitted lead information.
6. Lead data is automatically transferred to Mango CRM.
7. Sales or marketing teams manage and follow up with leads through the CRM.

## Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/marketing-growth-automation.git
cd marketing-growth-automation
```

### Install Dependencies

```bash
pip install flask
```

### Run Backend

```bash
python app.py
```

Backend Server:

```text
http://127.0.0.1:5000
```

### Run Frontend

Open:

```text
index.html
```

or use VS Code Live Server.
