# Marketing Growth Automation System

## Overview

This project is a complete marketing automation platform designed to generate, collect, and manage leads efficiently.

The system consists of:

* Responsive Marketing Website
* Zapier AI Chatbot Integration
* Google Form Lead Collection
* Automated Lead Transfer to Mango CRM
* Growth Strategy and Marketing Services Showcase

---

## Features

### Responsive Website

* Mobile-friendly design
* Interactive animations
* Smooth scrolling navigation
* Modern marketing agency UI

### Growth Mixer

* Interactive channel allocation simulator
* Dynamic reach forecasting
* Real-time visualization

### AI Chatbot

* Embedded Zapier chatbot
* Lead qualification
* Customer support automation
* Appointment scheduling

### Lead Collection

* Google Form integration
* User information capture
* Automated submission handling

### CRM Automation

* Zapier workflow automation
* Automatic lead transfer to Mango CRM
* Centralized lead management

## Workflow

User Visits Website
↓
User Interacts with Chatbot
↓
User Fills Google Form
↓
Google Form Submission
↓
Zapier Automation Trigger
↓
Lead Sent to Mango CRM
↓
Sales Team Follow-up
